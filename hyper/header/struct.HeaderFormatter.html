pub struct HeaderFormatter<'a, H: HeaderFormat>(pub &'a H);

A wrapper around any Header with a Display impl that calls fmt_header.

This can be used like so: format!("{}", HeaderFormatter(&header)) to get the representation of a Header which will be written to an outgoing TcpStream.
